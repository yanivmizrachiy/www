# כיסוי חקירת הכפתורים — /guide מול מסמך המקור

מקור: `מדריך_הכפתורים_מודל_משרד_החינוך` (Google Doc). לא נמצא קובץ `.txt` מקומי בריפו; מבנה המסמך וכל אזוריו אומתו דרך יצוא הטקסט של המסמך, ורשימת הפריטים המחייבת (15 הסעיפים) שימשה כמקור אמת. כל פריט מופה למפת הכפתורים במדריך — מקור אמת יחיד: `src/data/guideButtons.ts`.

סטטוסים: **COVERED** = מופיע ככפתור במפה · **MERGED_WITH_OTHER_SECTION** = הוסבר פעם אחת במקום אחר / אוחד עם פריט קרוב (מניעת כפילות) · **PARTIAL** · **BLOCKED**.

שפת ההסבר במדריך אחידה ופשוטה: "אם לוחצים — נפתח..." / "משם:" / "ראו גם".
