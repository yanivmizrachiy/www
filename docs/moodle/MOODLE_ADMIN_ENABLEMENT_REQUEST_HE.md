# בקשת בירור למנהל Moodle / משרד החינוך

שלום,

אנחנו מפתחים כלי מורה חיצוני הנפתח מתוך Moodle כ־External Tool / LTI, וכרגע הוא כבר מזהה בהצלחה את הקשר הקורס מתוך Moodle.

כדי לבדוק אם ניתן בעתיד להרחיב את האוטומציה בצורה רשמית, בטוחה ובהרשאות קריאה בלבד, נשמח לברר:

1. האם הכלי מוגדר אצלכם כ־LTI 1.1 או LTI 1.3.
2. אם זה LTI 1.3, האם LTI Advantage פעיל עבור הכלי.
3. האם NRPS פעיל עבור הכלי, ובאילו תנאי פרטיות.
4. האם AGS פעיל עבור הכלי, ואילו scopes מאושרים.
5. האם Moodle Web Services פעיל במערכת.
6. האם ניתן להקים חשבון שירות ייעודי עם הרשאות read-only בלבד.
7. האם ניתן להנפיק מפתח גישה מצומצם ולא גלובלי.
8. האם ניתן לאשר פונקציות קריאה כגון:

- core_webservice_get_site_info
- core_enrol_get_enrolled_users
- core_course_get_contents
- core_completion_get_activities_completion_status
- gradereport_user_get_grade_items
- core_grades_get_grades
- report_log_get_events

המטרה שלנו:

- לא לבצע כתיבה ל־Moodle
- לא לעקוף הרשאות
- לא לחשוף מידע מעבר לנדרש
- לבדוק אינטגרציה רשמית, מצומצמת ומאובטחת
- לא לשמור פרטי גישה בקוד או בריפו

נשמח גם לדעת אם יש מגבלות פרטיות על שיתוף:

- שמות
- אימיילים
- תפקידים
- מזהים פנימיים
