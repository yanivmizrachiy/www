# Product Scope

## מקור אמת
דף הכללים המלא: `docs/product/rules.md`

## למורה (תכולה פונקציונלית)
- דשבורד פתיחה ברור בעברית.
- תלמידים.
- משימות.
- ציונים.
- פעילות.
- יצוא.
- הגדרות.
- דוחות לפי תלמיד/קבוצה/טווח זמן.

## רק על בסיס נתונים אמיתיים
- זמן פעילות יומי.
- משימות וקישורים.
- ניסיונות.
- ציונים.
- אירועים.

## כללי עבודה קבועים
- אין דמו ואין נתונים מומצאים.
- אין פיצ'רים לא מחוברים מקצה לקצה.
- אין כפילויות וקבצים מיותרים.
- תיעוד תמיד מסונכרן עם המצב בפועל.

## סטטוס על-תמציתי
- קיים: תשתית שרת, Launch בסיסי, Dashboard RTL, API בסיסי, יצוא CSV.
- חסר: אינטגרציית Moodle מלאה, דוחות מתקדמים, יצוא מורחב, יכולות עריכה מלאות לפי הרשאות.
