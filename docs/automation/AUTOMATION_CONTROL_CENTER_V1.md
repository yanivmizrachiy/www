# Automation Control Center V1

הגרסה הראשונה של מרכז השליטה באוטומציה מציגה את מצב החיבור ל-Moodle ואת כיווני התקדמות האוטומציה, בלי להמציא נתונים.

## מה כולל

- דף חדש ביישום: `/automation`
- API backend חדש:
  - `GET /api/automation/capabilities`
  - `GET /api/automation/export-links`
- סטטוס חיבור Moodle / LTI
- זיהוי קורס ופרטי מורה/קורס
- בדיקת ייבוא קיים ל־Participants, Gradebook, Logs ו־Course Structure / Activity Completion
- קישורי דו"חות Moodle אמיתיים כאשר Course ID זוהה
- הודעה ברורה: סנכרון API מלא עדיין לא הופעל

## כללי אבטחה

- איננו מחזירים סודות.
- איננו משתמשים בסיסמות Moodle.
- אין החזרת שורות תלמידים.
- אין ייצור או הצגת נתונים מזויפים.
- איננו טוענים או מבצעים קריאות למודל Moodle כדי לבנות את הקישורים.

## מתי זה עובד

- כאשר המורה פותח את הכלי מתוך מרחב Moodle אמיתי.
- כאשר קיימת session LTI תקפה.
- כאשר ניתן לזהות Course ID מתוך ההקשר.
- כאשר כבר יובאו דוחות אמיתיים דרך הייבוא הקיים.

## מה עוד חסר

- אין עדיין שאיבת Moodle Web Services מאומתת.
- אין עדיין סנכרון רקע מלא.
- יש להמשיך להגדיר `MOODLE_WS_TOKEN` ולבדוק את המסלול האוטומטי.
