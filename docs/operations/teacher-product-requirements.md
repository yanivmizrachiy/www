# Teacher Product Requirements

The system is a real teacher product, not a demo.

## Dashboard

Must show real data only:

- teacher name
- Moodle course/space name
- connection status
- last update time
- activities by chapters when available
- tasks when chapters are unavailable

## Tasks

Real task categories only:

- online activities
- worksheets
- presentations
- quizzes / assignments when Moodle provides them

## Reports

When real data exists:

- grades by date
- grades by student
- grades by group
- grades by task
- grades by chapter

## Design

- Hebrew RTL
- premium colors
- clear contrast
- subtle 3D buttons
- no fake buttons
- no demo data

## Automation

Use the strongest real automation available:
LTI, NRPS, AGS, Moodle Web Services if real token exists, or smart real-report import fallback.
