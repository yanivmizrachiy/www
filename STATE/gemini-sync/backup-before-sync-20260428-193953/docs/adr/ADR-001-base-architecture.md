# ADR-001 בסיס המוצר

## החלטה
נשארים עם Node.js + Express + LTI 1.1 + Dashboard בעברית + Data Store מקומי לשלב bootstrap.

## למה
- כבר קיים בסיס עובד
- אין סיבה לזרוק את המערכת
- החסם העיקרי הוא ingress/launch אמיתי ו-Moodle integration
- אפשר להרחיב בהמשך בלי לשבור את השלד

## לא עכשיו
- מעבר לסטאק חדש
- המצאת נתונים
- בניית דמו
- תלות ב-tunnels זמניים כפתרון קבוע
